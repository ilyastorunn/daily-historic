Download Historiq today and discover the story of every day.

## Implementation Instructions

### Step 1: Open Google Play Console
1. Go to https://play.google.com/console
2. Select Historiq (com.ilyastorun.histora)
3. Navigate to "Grow" > "Store presence" > "Main store listing"

### Step 2: Enter Metadata
1. **App name (Title):** Paste `Historiq - Daily History & On This Day Events` into the "App name" field
2. **Short description:** Paste the short description above into "Short description"
3. **Full description:** Paste the full description above into "Full description"

### Step 3: Validate
- Real-time character counters appear next to each field — confirm all show green
- Use the preview panel on the right to see how the listing renders on a phone
- Confirm the first two lines of the full description read well in the collapsed state

### Step 4: Save and Publish
- Click "Save" (saves as draft)
- To publish: go to "Review and publish" and submit
- Title changes may trigger a brief Google review (typically under 24 hours)
- Short description and full description changes go live within 1–2 hours

**Important notes:**
- Google extracts search keywords from your title and description — there is no separate keyword field
- Emojis are permitted but used sparingly here to maintain the intellectual/premium tone
- Respond to all user reviews within 48 hours — review response activity is a confirmed Play Store ranking signal
- Avoid keyword stuffing: Google's algorithm penalizes unnatural repetition
