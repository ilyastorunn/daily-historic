# Subscription Localizations — Historiq Pro

**App:** Historiq - History Daily (`com.ilyastorun.histora`, ASC: `6759553478`)
**Ürünler:**
- `com.ilyastorun.histora.pro.monthly` → "Historiq Pro Monthly"
- `com.ilyastorun.histora.pro.annual` → "Historiq Pro Annual"

**Kısıtlamalar (ASC):**
- Display Name: max 30 karakter
- Description: max 45 karakter

**Not:** ASC desteklediği locale listesi sabit. Bazı ülkeler için kendi locale kodu yok (ZA, PH, PK, NG → en-GB / en-US kullanılır). UAE ve Suudi Arabistan aynı `ar-SA` locale'ini paylaşır. `sv-SE` değil `sv`; `ja-JP` değil `ja`; `ko-KR` değil `ko` kullanılır.

## ASC CLI Komutları

### Ön Kontrol

```bash
# Mevcut lokalizasyonları kontrol et
asc subscriptions localizations list --subscription-id MONTHLY_ID --paginate --output table
asc subscriptions localizations list --subscription-id ANNUAL_ID --paginate --output table
```
